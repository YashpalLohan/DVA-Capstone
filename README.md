# DVA Capstone Project

This repository contains the resources and documentation for the DVA Capstone project.

## Project Structure

- **Calculation and Pivot Table**: Excel files and other calculation resources.
- **Cleaned Dataset**: Processed data ready for analysis.
- **Dashboards**: Tableau/PowerBI dashboard files.
- **Documentation**: Project reports and guides.
- **Presentation**: Final presentation slides.
- **Raw Dataset**: Original, unmodified data sources.
